Installation & Setup

How do I install the Virtual Try-On extension? +

Installing the extension is simple:

  • Visit the Chrome Web Store
  • Click "Add to Chrome" on the Virtual Try-On extension page
  • Confirm by clicking "Add extension" in the popup
  • The extension icon will appear in your browser toolbar

Once installed, you can access the extension by clicking its icon or right-clicking on clothing images.

What permissions does the extension require? +

The extension requires minimal permissions:

  • contextMenus: To add the "Try On" option when you right-click on images
  • storage: To save your photos locally on your device
  • activeTab: To interact with the current webpage when you use the try-on feature

We only request permissions that are essential for the extension to function. Your data never leaves your device unless you connect to a third-party AI service.

Is the extension compatible with my browser? +

The Virtual Try-On extension is compatible with:

  • Google Chrome (version 88 and above)
  • Microsoft Edge (Chromium-based)
  • Brave Browser
  • Other Chromium-based browsers

Note: The extension is not compatible with Firefox or Safari at this time.

Usage & Features

How do I upload my photo for try-on? +

To upload your photo:

  • Click the Virtual Try-On extension icon in your browser toolbar
  • Click "Upload Photo" or drag and drop an image file
  • Supported formats: JPG, PNG, WebP (max 10MB)
  • For best results, use a clear, well-lit photo of yourself

Your photo is stored locally on your device and can be managed through the extension settings.

How do I try on clothing items? +

Once you've uploaded your photo:

  • Navigate to any online clothing store
  • Right-click on a clothing item image
  • Select "Try On with Virtual Try-On" from the context menu
  • Wait a few seconds for the AI to generate your try-on result

The try-on result will appear in a popup window showing how the item might look on you.

Which websites does the extension work with? +

The extension works with most major e-commerce websites including:

  • Amazon, eBay, Etsy
  • Fashion retailers like Zara, H&M, ASOS
  • Department stores like Nordstrom, Macy's
  • Any website with clothing images

The extension can detect clothing images on most websites. If you encounter issues with a specific site, please let us know.

How accurate are the try-on results? +

Try-on results are AI-generated approximations and should be considered as visual guides rather than exact representations. Accuracy depends on:

  • Quality of your uploaded photo
  • Clarity and angle of the clothing item
  • Complexity of the garment
  • AI model capabilities

Results are best for tops, dresses, and jackets. Complex patterns or very detailed items may not render perfectly.

API & Configuration

Do I need an API key to use the extension? +

Yes, you need to provide your own API key from a supported AI service provider. This approach:

  • Keeps costs transparent and under your control
  • Ensures your data only goes to services you trust
  • Allows you to choose your preferred AI provider
  • Eliminates the need for subscription fees

Supported providers include OpenAI, OpenRouter, and other compatible services.

How do I get an API key? +

To get an API key:

  • OpenAI: Visit platform.openai.com, create an account, and generate an API key
  • OpenRouter: Visit openrouter.ai, sign up, and create an API key
  • Add credits to your chosen provider's account
  • Copy the API key and paste it into the extension settings

Most providers offer free credits for new users to get started.

How much does it cost to use the extension? +

The extension itself is free. You only pay for:

  • API usage from your chosen AI provider
  • Typical cost: $0.01-$0.10 per try-on generation
  • Exact pricing depends on the AI model and provider you choose

This pay-per-use model means you only pay for what you actually use, with no subscription fees.

Is my API key secure? +

Yes, your API key security is prioritized:

  • API keys are stored locally in encrypted browser storage
  • Keys are never transmitted to our servers
  • Only sent directly to your chosen AI provider
  • You can delete or change your API key anytime

We recommend regularly rotating your API keys and monitoring usage through your provider's dashboard.

Privacy & Data

Where are my photos stored? +

Your photos are stored locally on your device:

  • Stored in your browser's local extension storage
  • Never uploaded to our servers
  • Only transmitted to AI providers when you generate try-ons
  • You can delete photos anytime through extension settings

This local-first approach ensures maximum privacy and control over your personal images.

What data is sent to AI providers? +

When you generate a try-on, only these items are sent:

  • Your uploaded photo (temporarily, for processing)
  • The clothing item image you selected
  • Basic processing parameters

No personal information, browsing history, or other data is transmitted. The AI provider processes your request and returns the try-on result.

Can I delete my data? +

Yes, you have full control over your data:

  • Delete individual photos through the extension popup
  • Clear all stored data in extension settings
  • Uninstall the extension to remove all local data
  • Contact AI providers directly about their data retention policies

Since data is stored locally, deletion is immediate and permanent.

Troubleshooting

The extension isn't working on a specific website +

If the extension doesn't work on a particular website:

  • Refresh the page and try again
  • Make sure you're right-clicking directly on the clothing image
  • Check if the website blocks extensions or uses heavy security measures
  • Try disabling other extensions temporarily

If problems persist, please report the website URL to our support team.

Try-on generation is failing or slow +

If try-on generation isn't working:

  • Check your API key is valid and has sufficient credits
  • Verify your internet connection is stable
  • Try with a different, clearer clothing image
  • Check if your AI provider is experiencing downtime

Generation typically takes 5-15 seconds. Longer times may indicate API or network issues.

The extension popup won't open +

If the extension popup isn't appearing:

  • Make sure the extension is enabled in chrome://extensions/
  • Try restarting your browser
  • Check if the extension icon is visible in the toolbar
  • Right-click the extension icon and select "Options" or "Manage"

If these steps don't help, try reinstalling the extension.
