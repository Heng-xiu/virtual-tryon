Common Issues

Installation & Setup

Extension not appearing in Chrome

Solution: Make sure you've enabled the extension in Chrome Extensions (chrome://extensions/). Look for the Virtual Try-On extension and ensure the toggle is switched on.

Right-click menu not showing "Try On" option

Solution: The try-on option only appears when right-clicking on images. Make sure you're right-clicking directly on a clothing item image, not on text or other page elements.

Can't upload photos

Solution: Check that the image file is in a supported format (JPG, PNG, WebP) and under 10MB in size. Also ensure you've granted the necessary permissions to the extension.

API & Configuration

API key not working

Solution: Verify that your API key is correctly entered without extra spaces. Make sure the key has sufficient credits and the correct permissions for the AI service you're using.

Try-on generation fails

Solution: This could be due to API limits, network connectivity, or image quality issues. Try with a different image or check your API service status.

Slow generation speed

Solution: Generation speed depends on your chosen AI provider and current server load. Consider upgrading your API plan or trying during off-peak hours.

Privacy & Data

How to delete stored photos

Solution: Open the extension popup, go to Settings, and use the "Clear Photos" button. You can also delete individual photos from the photo management section.

Extension permissions concerns

Solution: We only request necessary permissions: contextMenus (right-click menu), storage (local photo storage), and activeTab (interact with current page). No data is sent to our servers.
